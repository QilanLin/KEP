# ⚡ 快速入门指南

**5分钟快速开始使用Isabelle Sledgehammer Fuzzer**

---

## 🎯 目标

本指南将帮助您在5分钟内运行第一次fuzzing测试。

---

## ✅ 前置检查

### 1. 检查Python

```bash
python3 --version
```

**需要**: Python 3.8+

### 2. 检查Prover

```bash
which z3
which cvc5
```

**至少需要一个**: Z3 或 cvc5

### 3. 检查种子文件

```bash
ls ../sledgehammer_export/*.p | head -5
```

**需要**: 至少有一个`.p`文件

---

## 🚀 三步快速开始

### 第一步：进入目录

```bash
cd fuzzer
```

### 第二步：运行测试

```bash
python3 main.py --max-seeds 2 --num-mutants 3
```

### 第三步：查看结果

```bash
# 查看统计摘要（在终端输出中）
# 或查看详细统计
cat fuzzer_results/stats/stats.json | python3 -m json.tool
```

**完成！** 🎉
